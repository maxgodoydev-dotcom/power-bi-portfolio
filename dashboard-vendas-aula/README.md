## ✦ Sobre o projeto

Este dashboard foi desenvolvido em **Power BI** durante uma atividade acadêmica presencial, com foco na criação de painéis visuais para análise de **vendas**, **clientes**, **regiões** e **indicadores executivos**.

A proposta foi praticar a organização de dados em uma visualização clara, objetiva e útil para leitura gerencial.

Arquivo principal:

```txt
Dashboard_Vendas_Clientes_Regiao.pbix
````

---

## ✦ Objetivo

O objetivo deste estudo foi construir uma visão analítica em Power BI capaz de apoiar a leitura dos principais indicadores comerciais.

O dashboard busca transformar dados em informações visuais para facilitar:

* análise de vendas;
* leitura de desempenho comercial;
* comparação entre regiões;
* acompanhamento de clientes;
* visão executiva dos resultados.

---

## ✦ Indicadores trabalhados

| Área            | Indicadores                                  |
| --------------- | -------------------------------------------- |
| **Vendas**      | Volume de vendas, desempenho e comparações   |
| **Clientes**    | Distribuição, comportamento e participação   |
| **Regiões**     | Análise por localidade e desempenho regional |
| **Executivo**   | Visão resumida para tomada de decisão        |
| **Performance** | Comparações visuais e leitura de resultados  |
