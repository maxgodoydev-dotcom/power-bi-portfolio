## ✦ Sobre o projeto

Este dashboard foi desenvolvido em **Power BI** como parte do projeto acadêmico **Studio Patty Leão — Sistema Operacional de Salão**.

A proposta é representar, com dados genéricos e simulados, uma visão operacional de um salão de beleza, conectando informações de atendimentos, serviços, profissionais, produtos da vitrine e resultados financeiros.

Arquivo principal:

```txt
Studio_Patty_Leao_BI.pbix
````

---

## ✦ Objetivo

Criar uma visão gerencial simples e visual para apoiar a leitura da operação do salão.

O dashboard foi pensado para facilitar a análise de indicadores como faturamento, atendimentos, serviços, produtividade, cancelamentos e movimentação da vitrine.

---

## ✦ Indicadores trabalhados

| Área              | Indicadores                                                 |
| ----------------- | ----------------------------------------------------------- |
| **Financeiro**    | Faturamento, ticket médio e projeções                       |
| **Operação**      | Volume de atendimentos, serviços realizados e cancelamentos |
| **Profissionais** | Produtividade e desempenho por profissional                 |
| **Vitrine**       | Produtos, reservas e giro de itens                          |
| **Gestão**        | Visão gerencial para apoio à tomada de decisão              |

---

## ✦ Conceitos aplicados

| Conceito                  | Aplicação                                               |
| ------------------------- | ------------------------------------------------------- |
| **Business Intelligence** | Organização dos dados em painéis de leitura gerencial   |
| **KPIs**                  | Indicadores para acompanhamento da operação             |
| **Visualização de dados** | Uso de gráficos, cartões, filtros e páginas analíticas  |
| **Análise operacional**   | Leitura de atendimentos, serviços, vitrine e desempenho |
| **Tomada de decisão**     | Apoio visual para interpretação rápida dos resultados   |

## ✦ Relação com o projeto principal

Este dashboard complementa o sistema web **Studio Patty Leão — Sistema Operacional de Salão**, projeto desenvolvido com foco em gestão operacional, agendamentos, estoque, financeiro manual, vitrine de reservas e BI básico.

A proposta do painel é demonstrar como os dados da operação podem ser organizados visualmente para apoiar decisões simples do negócio.

---

## ✦ Observação

Os dados utilizados neste dashboard são **genéricos/simulados**, com finalidade acadêmica e demonstrativa.

Nenhum dado sensível de clientes, empresa ou terceiros foi utilizado neste arquivo.
