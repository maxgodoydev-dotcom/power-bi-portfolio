## 📌 Sobre o projeto

Este dashboard foi desenvolvido em **Power BI** como parte do projeto acadêmico **Studio Patty Leão — Sistema Operacional de Salão**.

A proposta é representar, com dados genéricos/simulados, uma visão de indicadores operacionais de um salão de beleza, conectando informações de atendimentos, serviços, profissionais, produtos da vitrine e resultados financeiros.
